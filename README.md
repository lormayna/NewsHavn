## Prerequisites

* [Go](https://go.dev/)
* [Task](https://taskfile.dev/)

## Run locally

1. Make sure you have set up the env variables (see `.env_example`)
2. Start DB
```bash
$ task db
```
3. Run
```bash
$ task run
```

The app should spin up on `localhost:8080`. 

For more options, run `task -l`
